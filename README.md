# worktree-spawn-skill

Spawn parallel git worktrees with session context for Claude Code and Codex.

## Structure

- `claude-code/` — Claude Code skill (SKILL.md, scripts, templates)
- `codex/` — Codex skill variant (SKILL.md, templates)

## Installation

Used as a git submodule in [my-skills](https://github.com/B0Qi/my-skills).
